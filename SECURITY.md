# セキュリティポリシー

## 脆弱性の報告

セキュリティ上の脆弱性を発見された場合は、**公開の Issue には投稿せず**、
以下の手順で非公開にご報告ください。

### 報告方法

GitHub の **Private vulnerability reporting** 機能を使用してください。

1. [bizprint のセキュリティページ](https://github.com/biz-Stream/bizprint/security) にアクセス
2. 「Report a vulnerability」をクリック
3. 脆弱性の詳細を記入して送信

### 報告に含めていただきたい情報

- 脆弱性の種類（例: リモートコード実行、情報漏洩、権限昇格など）
- 影響を受けるモジュール・ファイルのパス
- 再現手順
- 想定される影響範囲
- 可能であれば修正案

### 報告後の対応について

本プロジェクトはオープンソースソフトウェアであり、対応はベストエフォートで行います。
報告いただいた脆弱性については確認・修正に努めますが、対応時期や修正を保証するものではありません。

報告者の方には、修正完了まで脆弱性の詳細を非公開にしていただくようお願いいたします。
