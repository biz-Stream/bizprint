# コントリビューター行動規範

## 私たちの約束

メンバー、コントリビューター、リーダーとして、年齢、体格、障がいの有無、
民族、性別、性自認と性表現、経験の程度、学歴、社会的地位、国籍、容姿、
人種、カースト、肌の色、宗教、性的指向に関わらず、すべての人にとって
ハラスメントのないコミュニティへの参加体験を提供することを約束します。

私たちは、オープンで友好的、多様性を認め、包摂的かつ健全なコミュニティに
貢献する方法で行動し、交流することを約束します。

## 私たちの行動基準

前向きな環境を作り上げることに貢献する行動の例:

- 他の人に対する思いやりと親切を示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを提供し、また快く受け入れること
- 私たちのミスの影響を受けた人に対して責任を受け入れ、謝罪し、
  その経験から学ぶこと
- 個人としてだけでなく、コミュニティ全体にとって最善なことに焦点を
  あてること

参加者にとって受け入れられない行動の例:

- 性的な言葉や画像の使用、およびあらゆる種類の性的な注目やアプローチ
- 荒らし行為、侮辱的または軽蔑的なコメント、個人的または政治的な攻撃
- 公的または私的なハラスメント
- 住所やメールアドレスなど、他者の個人情報を明示的な許可なく公開すること
- 専門的な場において不適切と合理的に判断されるその他の行為

## 執行責任

コミュニティリーダーは、私たちが受け入れられる行動の基準を明確にし、
執行する責任があります。また、不適切、脅迫的、攻撃的、有害と判断された
行動に対し、適切かつ公正な是正措置を取ります。

コミュニティリーダーは、コメント、コミット、コード、Wiki の編集、
Issue、およびこの行動規範に沿っていないその他のコントリビューションを
削除、編集、または拒否する権利と責任を有します。また、適切な場合には
モデレーションの決定の理由を伝えます。

## 適用範囲

この行動規範は、すべてのコミュニティスペースで適用されます。また、
公的なスペースで個人がコミュニティを公式に代表している場合にも適用されます。

## 執行

暴力的、ハラスメント的、またはその他の受け入れられない行動の事例は、
GitHub の [Private vulnerability reporting](https://github.com/brainsellers-com/bizprint/security)
機能を通じて、執行責任を持つコミュニティリーダーに非公開で報告してください。

すべての苦情は迅速かつ公正にレビューおよび調査されます。

すべてのコミュニティリーダーは、事例の報告者のプライバシーとセキュリティを
尊重する義務があります。
