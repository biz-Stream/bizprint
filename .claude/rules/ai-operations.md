# AI 運用ポリシー

## コンテキスト管理

大きなタスクはセッション分割を提案し、進捗を auto memory に記録する。

## エラーの学習

ツール実行やコマンド実行でエラーが発生し、原因を特定して解決した直後に、再発防止パターンをフィードバックメモリー（`feedback_command_pitfalls.md`）に記録する。「解決したら即記録」を徹底する。個別エラーではなくカテゴリ別に集約し、重複は統合する。
