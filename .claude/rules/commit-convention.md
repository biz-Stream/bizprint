# コミットメッセージ規約

## プレフィックス（必須）

| プレフィックス | 用途 |
|---|---|
| `feat:` | 新機能追加 |
| `fix:` | バグ修正 |
| `chore:` | 設定変更・メンテナンス・ビルド関連 |
| `docs:` | ドキュメントのみの変更 |
| `refactor:` | リファクタリング（機能変更なし） |
| `test:` | テストの追加・修正 |

## フォーマット

```
<プレフィックス> <変更内容の要約> (#<イシュー番号>)
```

- 要約は日本語で簡潔に記述する
- イシュー番号は `(#123)` 形式で末尾に付与する
- 例: `fix: PDF出力時のフォント埋め込みエラーを修正する (#45)`

## Co-Authored-By

Claude Code がコミットする場合は、コミットメッセージの末尾に `Co-Authored-By` を付与する。
モデル名はセッションごとに異なるため、システムプロンプトに記載されている実際のモデル名を使用すること。

```
Co-Authored-By: <モデル名> <noreply@anthropic.com>
```
